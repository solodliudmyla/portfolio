The website receive data used the OMDb API. (OMDb is the Open Movie Database. The OMDb API is a RESTful web service to obtain movie information.) User should write a word and get movies result contain the word in titles. I used Swiper library to create touch slider. To navigate user can use touch screen, arrows at keyboard, mouse for arrows on screen. I built the website view according to my taste and understanding UX.

I used JavaScript for:
